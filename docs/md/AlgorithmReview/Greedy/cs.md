# Task Assignment [Medium]

- You're given an integer `k` representing a number of workers and an array of positive integers representing durations of tasks that must be completed by the workers. Specifically, each worker must complete two unique tasks and can only work on one task at a time. The number of tasks will always be even and each task will have a corresponding positive integer indicating its duration. You can assume that multiple workers will always be available to complete a task and a worker can be assigned two tasks if and only if the sum of their durations is less than or equal to the maximum duration that a worker is allowed to work.
- Write a function `taskAssignment` that returns a list of pairs, where each pair represents the tasks assigned to a particular worker. The pairs should be in the following format: `[task1, task2]`, where the order of `task1` and `task2` doesn't matter. The function should return the pairs in any order.

**Sample Input**

```
k = 3
tasks = [1, 3, 5, 3, 1, 4]
```

**Sample Output**

```
[[0, 2], [4, 5], [1, 3]]
```

# 面试阶段


- 最近刚参加了一个外企的面试，软件开发职位。技术面是使用英语的部分基本就是自我简介和简单介绍一些做过的项目，这些都在邮件中通知了，而这些都大概是可以事先准备好的(强烈建议先把要说的写在纸上，没必要逐字逐句写，可以大概列个提纲，这样心里会放松很多)，之后就会切换到中文了。而在后来HR面的时候也是中间用了一段英文交流，主要是问一些个人方面的问题（比如欣赏哪类人，自己的性格如何）。总的来说绝大部分是用中文的，英文只要能让对方明白你的意思即可。不要紧张，可以慢慢说，不是说在想不起一个词的时候就卡那里想个半天，这样给人印象不好，是要在说之前理好自己的思路，思路清晰了表达就轻松了。（建议可以带一瓶水放在旁边，如果觉得比较紧张，就亲抿一口，此举个人觉得有帮助）。毕竟面试的是开发岗位，最重要的还是自己的技术，英语就是能沟通就好了吧，即使可能要多说几句才能让人明白本可一句说明的话。（但英语很好的话当然是加分项

# BQ模版

#### What if teammates / colleagues don't work / are difficult to get along with?

一个项目每个人都有擅长/喜欢的部分 - 所以我会尽量在队友喜欢/擅长的方向和自己喜欢/擅长的方向寻找互补之处，最优化项目分工。如果依然效率低，那么与其去提供负面反馈，我更倾向于先想办法了解where they are coming from，了解他们在工作/交流上的road-blocker是什么。

#### 谈一谈你帮助你同事或者朋友或者任何一个人取得进步的一次经历

这类问题就是看你作为senior，如何mentor junior，如果你确实有类似经历，并且你是一个会思考会总结的senior，那么你最好说一些方法论。我这里提方法论，并不是说你得多擅长mentor别人，而是证明你mentor过别人并且你爱思考爱总结，所以你都已经弄出方法论了。有点抽象，举个例子，你mentor组里一个暑期intern，本来ta只需要做一个60分的项目，就达到往年组里对于暑期intern期待了。你经验丰富，组里老司机，帮助他定位了一个更ambitious的项目，并且引导ta制定了详细的执行计划。最终，暑期实习结束，你intern的项目大受好评，他自信满满，觉得在这个组的这个暑假成长巨大，提升了自己解决更复杂问题的能力，balabala。

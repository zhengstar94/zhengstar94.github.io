# 📇CheckList  | 出发前检查单


- 预约前往机场（火车站）的车


- 设定次日出行起床闹钟（红眼航班）


- 拔掉家用电器插头


- 关闭强电箱电源回路（冰箱所在回路保持打开）


- 关闭天然气入户阀门


- 锁紧窗户（纱窗）、关窗帘


- 根据需要室内留一盏灯


- 清理冰箱内的剩菜剩饭


- 清理并倒掉室内所有垃圾


- 再次确认行李准备齐全


- 再次确认手机、钱包、身份证、钥匙


- 反锁入户门
